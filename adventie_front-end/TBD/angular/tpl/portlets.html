Pages Portlets

Effortless Customization


Portlets are pluggable UI components that are managed and displayed in a web portal. Portlets in Pages are created by reusing the panels introduced in Bootstrap to enable effortless customization.

Advance Tools

We have crafted Pages Portlets to suit any use case. Add a maximize button into your portlets controls bar to make the portlets go full-screen. This will come handy if you want to show lot of content inside a portlets and want to give the content some room to breath

Basic Tools

Basic Portlet tools include a slide toggle button , refresh button and a close button All these are fully customizable and come with callback functions to integrate with your code. Clicking on the refresh button will simulate an AJAX call.

SVG animated progress indicators


Pages comes with two types of inderminate progress indicators: linear and circular. These have been created using animated SVGs to support various screen sizes and retina displays. We have integrated them into portlets too.


More about progress & activity indicators

Portlet Title

Minimal Portlet

Click on the refresh icon to simulate an AJAX call and to see an animated circular progress indicatorabove the portlet. This is the Pages default progress indicator for portlets. Don't like this style? Simply change the style by setting portlet options.

Portlet Progress indicators
Portlet Title

Linear Progress

Click on the refresh icon to simulate an AJAX call and to see an animated progress bar indicator above the portlet. These progress bars come in seven different colors that are available in the Pages contextual color scheme.

Portlet Title

Circular Progress

Portlet with Errors
